Hello and welcome,

to run these maps please use a unity version that is "2017" or newer

To run these programs please move them out of the zip file, and open the unity scene inside the Assets folder.
click on run to run the map.
click anywhere on the map, and the console will let you know if you clicked on the map "hit" or outside it "not"
profiler tab will show an overview of the efficiency for the test

Enjoy!!
